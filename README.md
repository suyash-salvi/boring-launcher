# Boring Launcher 📱

A minimalist, intentional, and open-source Android launcher built with Flutter. Designed to reduce screen time and help you stay mindful of your digital habits.

## ✨ Features

- **Minimalist Interface**: No app icons, no colors, no distractions. Just text.
- **Intentional Friction**: Mark apps as "Distracting" to receive a mindful nudge before opening them.
- **Customizable Home Screen**: Easily select and reorder the apps you use most.
- **Searchable Apps**: Quickly find and launch any app on your device.
- **No Tracking**: Your app usage data never leaves your device.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Android Studio or VS Code
- An Android device (Android 7.0+)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/boring-launcher.git
   ```
2. Navigate to the project directory:
   ```bash
   cd boring-launcher
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
